Create an account
=================

You can create an account on DeepSource using an existing GitHub, GitLab, Bitbucket, or Azure DevOps Services account. Once you have signed up, DeepSource will ask for your permission to install the DeepSource app for that provider in your account.

### 

Requirements

[](#requirements)

*   An existing GitHub/GitLab/Bitbucket/Azure DevOps Services account.
*   Owner permissions for the repository.

Connect GitHub

[](#connect-github)
=====================================

1.  On the DeepSource Sign-up page click "Continue with GitHub".
2.  If you aren't already logged into GitHub, you will be asked to enter your login credentials to proceed.
3.  Accept the required permissions.
4.  Install the DeepSource App on GitHub.

![](https://files.readme.io/eb9ef23-github.png)

5.  From the list, select the account (user, or organization) for which you want to enable DeepSource and click "Install".
6.  After choosing an appropriate payment plan, you will be redirected to the dashboard.

Connect GitLab

[](#connect-gitlab)
=====================================

1.  On the DeepSource Sign-up page, click "Continue with GitLab".
2.  If you aren't already logged into GitLab, you will be asked to enter your login credentials to proceed.
3.  Accept the required permissions.

![](https://files.readme.io/7f3e772-gitlab-1.png)

4.  From the list, select the account (Personal or Team) for which you want to enable DeepSource.

![](https://files.readme.io/5eee6f7-SCR-20231017-llud.png)

5.  After you choose an appropriate payment plan, you will be redirected to the dashboard.

Connect Bitbucket

[](#connect-bitbucket)
===========================================

1.  On DeepSource Sign up page click "Continue with Bitbucket".
2.  If you aren't already logged into Bitbucket, you will be asked to enter your login credentials to proceed.
3.  Accept the required permissions.
4.  Install the DeepSource App on Bitbucket. From the "Authorize for" list, select the account (user, or organization) for which you want to enable DeepSource and click "Grant Access".

![](https://files.readme.io/25377bf-bitbucket-1.png)

5.  After you choose an appropriate payment plan, you will be redirected to the dashboard.

Connect Azure DevOps Services

[](#connect-azure-devops-services)
===================================================================

1.  On the DeepSource Sign-up page, click "Continue with Azure DevOps Services".
2.  If you aren't already logged into Azure DevOps Services, you will be asked to enter your login credentials to proceed.
3.  Accept the required permissions.

![](https://files.readme.io/c88fbfe-ads-1.png)

4.  Select the account (Azure DevOps Organization) for which you want to enable DeepSource and click "Install".

![](https://files.readme.io/916cc21-SCR-20231017-lnjo.png)
