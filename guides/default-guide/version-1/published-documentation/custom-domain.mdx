Set a custom domain
===================

Learn how to set a custom domain for your GitBook organization or Docs sites

Last updated 5 months ago

By default, your sites are accessible on a `[subdomain].gitbook.io` domain.

You can customize this by setting a custom domain, meaning your audience can access your documentation on a chosen domain.

**Permissions**

**Organization-level** domains can be set by users with admin rights. **Site** domains can be set by users with creator or admin rights.
