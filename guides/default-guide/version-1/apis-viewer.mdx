Viewer
======

viewer

[](#viewer)
---------------------

Lookup the details of the authenticated user.

### 

Example Request

[](#example-request)

GraphQL

`query {   viewer {     id     firstName     lastName     email   } }`

### 

Example Response

[](#example-response)

GraphQL

`{   "data": {     "viewer": {       "id": "VXNlcjp6a2dkZXk=",       "firstName": "John",       "lastName": "Doe",       "email": "",     }   } }`
