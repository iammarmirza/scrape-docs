Transformer
===========

transformer

[](#transformer)
-------------------------------

`Query: transformer` Fetch details of a transformer using the transformer shortcode.

### 

Arguments

[](#arguments)

Field

Type

Description

shortcode

[String](https://deepsource-docs.readme.io/docs/apis-reference-scalars#string)!

Shortcode of the transformer.

### 

Sample Request

[](#sample-request)

GraphQL

`query {   transformer(shortcode:"gofmt"){    	id     name     shortcode   } }`

### 

Sample Response

[](#sample-response)

JSON

`{   "data": {     "transformer": {       "id": "VHJhbnNmb3JtZXI6bGp6a3d6",       "name": "Go fmt",       "shortcode": "gofmt"     }   } }`

transformers

[](#transformers)
---------------------------------

`Query: transformer` Fetch details of all available transformers on DeepSource.

### 

Arguments

[](#arguments-1)

Field

Type

Description

offset

[Int](https://deepsource-docs.readme.io/docs/apis-reference-scalars#int)

before

[String](https://deepsource-docs.readme.io/docs/apis-reference-scalars#string)

after

[String](https://deepsource-docs.readme.io/docs/apis-reference-scalars#string)

first

[Int](https://deepsource-docs.readme.io/docs/apis-reference-scalars#int)

last

[Int](https://deepsource-docs.readme.io/docs/apis-reference-scalars#int)

### 

Sample Request

[](#sample-request-1)

GraphQL

`query {   transformers{    	edges {       node {         id         name         shortcode       }     }   } }`

### 

Sample Response

[](#sample-response-1)

JSON

`{   "data": {     "transformers": {       "edges": [         {           "node": {             "id": "VHJhbnNmb3JtZXI6ZXJ6amFi",             "name": "dotnet-format",             "shortcode": "dotnet-format"           }         },         {           "node": {             "id": "VHJhbnNmb3JtZXI6bGp6a3d6",             "name": "Go fmt",             "shortcode": "gofmt"           }         }       ]     }   } }`
