clang-format
============

Transform all incoming C and C++ code with [clang-format](https://github.com/llvm/llvm-project/tree/main/clang/tools/clang-format). Documentation for clang-format configurations can be found at [clang-format docs](https://clang.llvm.org/docs/ClangFormat.html).
