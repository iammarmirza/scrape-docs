Live edits
==========

Collaborate on pages in real-time with your teammates

**Live editing** is the default editing mode for any newly-created GitBook space. A space in live edit mode is editable by anyone with the .

With live editing enabled, you can see the avatars of anyone currently viewing the space in the top-right corner.

GitBook supports live collaboration, meaning you’ll be able to work on the same document with multiple members at the same time.

[](#toggling-live-edit-mode)

Toggling live edit mode


---------------------------------------------------------

You can lock or unlock live edits on a space from the Actions menu.

### 

[](#when-is-live-editing-not-available)

When is live editing _not_ available?

You cannot unlock live editing if:

1.  a space is with the **In collection**, **Public**, or **Unlisted** . We know this is a limitation, and we hope to change this in the future.
    
2.  a space has enabled.
    

Only users with the **admin** or **creator** can lock or unlock live edits.
