Formatting your content
=======================

You can format your content in a number of ways, using the context menu or keyboard shortcuts in the GitBook editor.

To format your text, simply select it and choose one of the formats from the context menu — or use a keyboard shortcut or Markdown syntax. We’ve listed those out below:

![](https://docs.gitbook.com/~gitbook/image?url=https%3A%2F%2F1050631731-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FNkEGS7hzeqa35sMXQZ4X%252Fuploads%252Fxca3kY7BI48kDVljwNXP%252Feditor-format.png%3Falt%3Dmedia%26token%3D85419909-8a7d-4b5b-aa42-7be9292f5452&width=768&dpr=4&quality=100&sign=bde10f25&sv=1)

bold, italics context menu with text highlighted

We’ve written these shortcuts using Mac keys. Use **Control** in place of **⌘ (Command)** on Windows or Linux operating systems. Check out to see all the shortcuts for all operating systems.

### 

[](#bold)

Bold

Keyboard shortcut: **⌘ + B**

Markdown

Copy

    **Bold**

### 

[](#italic)

Italic

Keyboard shortcut : **⌘ + I**

Markdown

Copy

    _Italic_

### 

[](#strikethrough)

Strikethrough

Keyboard shortcut: **⇧ + ⌘ + S**

Markdown

Copy

    ~~Strikethrough~~

### 

[](#code)

Code

Keyboard shortcut: **⌘ + E**

Markdown

Copy

    `Code`
