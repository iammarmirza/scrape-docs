SAML SSO & SCIM: Google Workspace

Configuring SAML SSO on Google Workspace

[](#configuring-saml-sso-on-google-workspace)
-----------------------------------------------------------------------------------------

For now, an admin (on Google Workspace) needs to create a custom SAML integration for DeepSource Enterprise. The steps for which are as given:

1.  Navigate to [https://admin.google.com/ac/apps/unified](https://admin.google.com/ac/apps/unified).
2.  On the menu bar, click on **"Add App" -> "Add custom SAML app"**.

![](https://files.readme.io/d338028-image.png)

3.  Fill in the following details:

![](https://files.readme.io/9497133-image.png)

Field

Value

App Name

DeepSource

4.  Click **"Continue"** again.
5.  Assuming that DeepSource is hosted on-premise at `https://deepsource.foobar.com`, fill in the following details accordingly:

> üöß
> 
> Note for Enterprise Cloud Users
> 
> 
> -------------------------------------
> 
> If you're on DeepSource Enterprise Cloud, replace [https://deepsource.foobar.com](https://deepsource.foobar.com) with [https://app.deepsource.com](https://app.deepsource.com)

![](https://files.readme.io/9e0fc91-image.png)

Field

Values

`https://deepsource.foobar.com/saml2/acs/`

`https://deepsource.foobar.com/saml2/metadata/`

Yes

5.  In "Attributes", add the following:  
    ![](https://files.readme.io/2eb12bc-image.png)

Google Directory attributes

App attributes

First name (choose from dropdown)

`first_name`

Last name (choose from dropdown)

`last_name`

6.  On the next screen, click on **‚ÄúDOWNLOAD METADATA‚Äù** button. A modal will open up, clicking on **‚ÄúDOWNLOAD METADATA‚Äù** button on the modal will ask you to download and save a `GoogleIDPMetadata.xml` file on your system.

![](https://files.readme.io/13a8d65-image.png) ![](https://files.readme.io/daaedcd-image.png)

7.  The file downloaded in the previous step should be hosted on a publicly accessible URL. You can either host it on your own infrastructure or an easy alternative is to paste it into a public gist on [https://gist.github.com/](https://gist.github.com/).
