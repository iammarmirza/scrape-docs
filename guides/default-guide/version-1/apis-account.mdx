Account
=======

The `Account` query is used to fetch the details of your team or individual account on DeepSource. It also provides mutations to configure account-wide settings.

account

[](#account)
-----------------------

`Query:account` Fetch the details of a single DeepSource account.

### 

Arguments

[](#arguments)

Field

Type

Description

login

[String](https://deepsource-docs.readme.io/docs/apis-reference-scalars#string)!

The login or username to lookup the account by.

vcsProvider

[VCSProvider](https://deepsource-docs.readme.io/docs/apis-reference-enums#vcsprovider)!

VCS Provider of the account.

### 

Sample Request

[](#sample-request)

GraphQL

`query {   account(login: "deepsourcelabs", vcsProvider: GITHUB) {     login     id     type     vcsProvider     vcsUrl     avatarUrl   } }`

### 

Sample Response

[](#sample-response)

JSON

`{   "data": {     "account": {       "login": "cyberdyne",       "id": "QWNjb3VudDp6bWpws2I=",       "type": "TEAM",       "vcsProvider": "GITHUB",       "vcsUrl": "https://github.com/cyberdyne",       "avatarUrl": "https://user-avatars.deepsource.io/demo-image.png"     }   } }`
