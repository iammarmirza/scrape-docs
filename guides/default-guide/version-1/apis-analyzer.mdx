Analyzer
========

analyzer

[](#analyzer)
-------------------------

`Query: analyzer` Fetch details of a single analyzer using the shortcode.

### 

Arguments

[](#arguments)

Field

Type

Description

shortcode

[String](https://deepsource-docs.readme.io/docs/apis-reference-scalars#string)!

Shortcode of the analyzer you'd like to get.

### 

Sample Request

[](#sample-request)

GraphQL

`query {   analyzer(shortcode:"python") {     id     shortcode     name   } }`

### 

Sample Response

[](#sample-response)

JSON

`{   "data": {     "analyzer": {       "id": "QW5hbHl6Zdd6bGtiZXZ6",       "shortcode": "python",       "name": "Python"     }   } }`

analyzers

[](#analyzers)
---------------------------

`Query: analyzers` Fetch all available analyzers on DeepSource.

### 

Arguments

[](#arguments-1)

Field

Type

Description

offset

[Int](https://deepsource-docs.readme.io/docs/apis-reference-scalars#int)

before

[String](https://deepsource-docs.readme.io/docs/apis-reference-scalars#string)

after

[String](https://deepsource-docs.readme.io/docs/apis-reference-scalars#string)

first

[Int](https://deepsource-docs.readme.io/docs/apis-reference-scalars#int)

last

[Int](https://deepsource-docs.readme.io/docs/apis-reference-scalars#int)

### 

Sample Request

[](#sample-request-1)

GraphQL

`query {   analyzers{    	edges {       node {         id         name         shortcode       }     }   } }`

### 

Sample Response

[](#sample-response-1)

JSON

`{   "data": {     "analyzers": {       "edges": [         {           "node": {             "id": "QW5hbHl6ZXI6bGtiZXZ6",             "name": "Python",             "shortcode": "python"           }         },         {           "node": {             "id": "QW5hbHl6ZXI6cmx6b2xi",             "name": "Go",             "shortcode": "go"           }         }       ]     }   } }`
