Repository
==========

Lookup a repository and associated information.

repository

[](#repository)
-----------------------------

`Query:repository` Fetch the details of one of your repositories.

### 

Arguments

[](#arguments)

Field

Type

Description

name

[String](https://deepsource-docs.readme.io/docs/apis-reference-scalars#string)!

The name of the repository to lookup.

login

[String](https://deepsource-docs.readme.io/docs/apis-reference-scalars#string)!

The login or username of the account under which the repository exists.

vcsProvider

[VCSProvider](https://deepsource-docs.readme.io/docs/apis-reference-enums#vcsprovider)!

VCS Provider of the account.

### 

Sample Request

[](#sample-request)

GraphQL

`query {   repository(       name:"demo-go",       login:"cyberdyne",       vcsProvider:GITHUB     ) {     name     defaultBranch     dsn     isPrivate     analysisRuns {       edges {         node {           runUid         }       }     }   } }`

### 

Sample Response

[](#sample-response)

JSON

`{   "data": {     "repository": {       "name": "demo-go",       "defaultBranch": "master",       "dsn": "https://",       "isPrivate": true,       "analysisRuns": {         "edges": [           {             "node": {               "runUid": "5d821174-04vf-4214-b56d-ced0a7ca209e"             }           },           {             "node": {               "runUid": "6ff1cdb8-9abe-4307-a177-4b9f8d3ba639"             }           },         ]       }     }   } }`

updateRepositoryDefaultBranch

[](#updaterepositorydefaultbranch)
-------------------------------------------------------------------

`Mutation:updateRepositoryDefaultBranch` Update a repository's default branch for baseline. If the repository is activated, this action will trigger a new analysis. Only available to users with `WRITE` permission on the repository.

### 

Arguments

[](#arguments-2)

Field

Type

Description

**input**

[UpdateRepositoryDefaultBranchInput](#updaterepositorydefaultbranchinput)!

Input for `UpdateRepositoryDefaultBranch` mutation

### 

Sample Request

[](#sample-request-2)

GraphQL

`mutation ($input: UpdateRepositoryDefaultBranchInput!) {   updateRepositoryDefaultBranch(input: $input) {     ok     repository {       defaultBranchName     }   } }`

JavaScript

`variables = {   "input": {     "id": "VHJhbnNmb3JtZXI6bGp6a3d6",     "defaultBranchName": "add-awesomeness"   } }`

### 

Sample Response

[](#sample-response-2)

JSON

`{     "updateRepositoryDefaultBranch": {     "ok": true,     "repository": {"defaultBranchName": "add-awesomeness"}   }   }`
