TeamMember

Represents a user within a team.

Field

Type

Description

**id**

[ID](https://deepsource-docs.readme.io/docs/apis-reference-scalars#id)!

The ID of the object.

**user**

[User](https://deepsource-docs.readme.io/docs/apis-viewer#user)!

The User instance.

**role**

[TeamMemberRole](https://deepsource-docs.readme.io/docs/apis-reference-enums#teammemberrole)!

The role this user has in the team.

**isOwner**

[Boolean](https://deepsource-docs.readme.io/docs/apis-reference-scalars#boolean)!

Whether this user is the owner of the team.

**joinedAt**

[DateTime](https://deepsource-docs.readme.io/docs/apis-reference-scalars#datetime)!

The time when this user joined the team.
