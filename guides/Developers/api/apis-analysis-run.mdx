Analysis Run
============

A `Run` is an analysis done against your codebase.

run

[](#run)
---------------

`Query: run` fetch the details of a single run using either the `runUid` identifier or `commitOid`.

### 

Arguments

[](#arguments)

Field

Type

Description

runUid

[UUID](https://deepsource-docs.readme.io/docs/apis-reference-scalars#uuid)

UID of the Analysis Run you want to get.

commitOid

[String](https://deepsource-docs.readme.io/docs/apis-reference-scalars#string)

Commit OID of the Analysis Run you want to get.

### 

Sample Request

[](#sample-request)

GraphQL

`query {   run(runUid:"0613332b-436a-429b-95e3-6e01d72208b3") {     id     createdAt     branchName     baseOid     finishedAt   } }`

or,

GraphQL

`query {   run(commitOid:"6df7f08b029abbf25905abf1bc79fde72934045b") {     id     createdAt     branchName     baseOid     finishedAt   } }`

### 

Sample Response

[](#sample-response)

JSON

`{   "data": {     "run": {       "id": "QW5hbHlzaXNadW46eHpkZ3Zi",       "createdAt": "2019-01-07T20:52:19.491314+00:00",       "branchName": "foobar",       "finishedAt": "2019-01-07T20:52:34.677357+00:00"     }   } }`

ChangesetStatsCounts

[](#changesetstatscounts)

Overall and newly added number of lines (or branches or conditions) in a changeset. Note: `0` depicts no lines (or branches or conditions) were found whereas `None` depicts the information is not available.

Field

Type

Description

**overall**

Overall number of lines (or branches or conditions) across the repository.

**overallCovered**

Overall number of lines (or branches or conditions) that are covered across the repository.

**new**

Newly added number of lines (or branches or conditions) in the changeset.

**newCovered**

Newly added number of lines (or branches or conditions) that are covered in the changeset.
