Prettier
========

Run [Prettier](https://github.com/prettier/prettier) to format all incoming JavaScript, TypeScript, Flow, JSX, JSON, CSS, SCSS, Less, HTML, Vue, Angular, GraphQL, Markdown and YAML code.
