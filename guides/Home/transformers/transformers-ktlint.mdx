Ktlint
======

Transform all incoming Kotlin code with [Ktlint](https://pinterest.github.io/ktlint/0.50.0/). Documentation for Ktlint's `.editorconfig` configuration can be found [here](https://pinterest.github.io/ktlint/0.50.0/rules/configuration-ktlint/).

=======================================================================

This section covers configuration specific to the `Ktlint` transformer. Please make sure to read the [general configuration guide](https://deepsource-docs.readme.io/docs/configuration) first.

name

[](#name)
-----------------

**Type:** String  
**Presence:** mandatory  
**Description:** Shortcode of this transformer.  
**Example:**

TOML

`name = "ktlint"`

enabled

[](#enabled)
-----------------------

**Type:** Boolean  
**Presence:** optional  
**Description:** Toggle whether this transformer should be run.  
**Example:**

TOML

`enabled = true`
